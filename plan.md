## Основные элементы реализации:

1. **Структура YDB таблиц**: users, organizations, memberships, plans, subscriptions, subscription_history, email_logs с необходимыми индексами

2. **JWT токены** со структурой: sub, user_id, email, role, org_id, exp, iat

3. **RBAC система** с ролями admin/manager/user и детальными правами для каждой операции

4. **Email верификация** через Yandex Cloud Postbox с комбинированным подходом (6-значный код + ссылка)

5. **Структура проекта** с пакетами internal/ для auth, rbac, storage, email, ydb, logger, grpc

6. **Дополнительные переменные окружения** для Postbox и JWT
