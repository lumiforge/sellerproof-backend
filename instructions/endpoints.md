## logout

`POST /api/v1/auth/logout` нужен для выхода пользователя из системы с одновременным отзывает refresh токена, который используется для обновления access токена. Это важно для прекращения возможности пользователя получать новые access токены после выхода, то есть для безопасности и контроля сессии.

Иными словами, этот endpoint:

- Прекращает аутентифицированный сеанс пользователя.
- Делает недействительным ранее выданный refresh токен, предотвращая его дальнейшее использование для получения новых access токенов.
- Обеспечивает корректное "завершение" сессии, снижая риск несанкционированного доступа при компрометации токенов.

Такой подход помогает улучшить безопасность, предотвращая использование украденных или скомпрометированных refresh токенов после выхода пользователя из системы.
